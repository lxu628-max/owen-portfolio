# Owen Portfolio 部署指南

## 方式一：Railway 部署（推荐，最快上线）

### 第一步：准备 GitHub
1. 登录 github.com
2. 点右上角 "+" → "New repository"
3. 仓库名填 `owen-portfolio`，选 Private，点 Create

### 第二步：安装 Git（如已安装跳过）
- 下载：https://git-scm.com/download/win
- 安装时全部默认选项即可

### 第三步：推送代码到 GitHub
1. 把 `owen-portfolio` 文件夹放到你喜欢的位置（比如桌面）
2. 在该文件夹上右键 → "Open Git Bash here"（或打开终端 cd 到该目录）
3. 依次执行以下命令：
```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/你的用户名/owen-portfolio.git
git push -u origin main
```
（把"你的用户名"替换成你的 GitHub 用户名）

### 第四步：部署到 Railway
1. 打开 https://railway.app ，用 GitHub 账号登录
2. 点 "New Project" → "Deploy from GitHub repo"
3. 选择 `owen-portfolio` 仓库
4. 等待自动部署完成（约2-3分钟）
5. 在 Settings → Networking → Generate Domain 生成公网域名
6. 访问生成的域名即可看到网站

### 第五步：修改后台密码
1. 访问 `你的域名/admin`
2. 用 admin / admin123 登录
3. 在后台"设置"中修改密码

---

## 方式二：本地运行（适合测试）

### 安装 Node.js
1. 下载：https://nodejs.org （选 LTS 版本）
2. 安装，全部默认

### 运行项目
```bash
cd owen-portfolio
npm install
node server.js
```
浏览器打开 http://localhost:3000

---

## 注意事项
- 默认管理员账号：admin / admin123（上线后务必修改）
- Railway 免费版重启后数据库会重置，建议升级为付费版（$5/月）开启持久存储
- 图片上传后存在服务器，持久存储开启后图片不会丢失
